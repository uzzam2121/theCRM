## Installation

```bash
npm install
```

## Development

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Deploy to Vercel

1. Push your code to GitHub
2. Import the project to Vercel
3. Vercel will automatically detect the Vite configuration
4. Deploy!

Or use the Vercel CLI:

```bash
npm i -g vercel
vercel
```
